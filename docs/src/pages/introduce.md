# 介绍

> `Dynamic Form` 是基于 `Vue3` 开发的一套专注于表单的组件，不是大而全的 UI 组件库，主要通过配置的形式使用表单，尽可能达到不写代码实现一个表单的创建，类似于之前基于 `Vue2 + Vant2` 开发的动态表单模式，[具体链接](https://github.com/D-xuanmo/v-form)，功能会更强大。
